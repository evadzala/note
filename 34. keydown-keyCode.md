### 紀錄內容：

因為遇到需要阻擋非英數的輸入，我的想法是從案件按下去偵測輸入的key，測試了一下input的keydown事件

```html
<input
	v-model:value="scanData.checkpoint_id"
	@keydown="keydownId"
/>
```

```jsx
const keydownId= (e: any): void => {
  console.log('keydownId', e.keyCode)
  const key = e.keyCode

	// 非英數的輸入(例如中文輸入)，keyCode都會為229
  if (key >= 229) {
    e.preventDefault() // 阻止這個鍵被輸入
  }
}
```
