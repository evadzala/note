### 紀錄內容：

有時候我們會希望使用者填表單時按照順序
前端可運用一些小技巧做控制
也方便使用者填表單
以下範例是讓使用者填完一個input後會自動跳至下一個

```html
<templete>
	<div>
		<input
			ref="nameInput" 
			v-model="data.name"
			@keyup.enter="goTOAge"
		/>
		<input 
			ref="ageInput" 
			v-model="data.age"
			@keyup.enter="goToPhone"
		/>
		<input 
			ref="phoneInput" 
			v-model="data.phone"
			@keyup.enter="Done"
		/>
	</div>
</template>

<script setup lang="ts">
import {ref} from "vue";

const nameInput = ref(null) as any
const ageInput = ref(null) as any
const phoneInput = ref(null) as any

const data = ref({
	name: '',
	age: '',
	phone: '',
})

const goToAge = () => {
	nameInput.value.blur() // 失焦input
	ageInput.value.focus() // 聚焦下一個input
}

const goToPhone = () => {
	ageInput.value.blur()
	phoneInput.value.focus()
}

const done = () => {
	console.log(data.value)
// 	call api
}
</script>
```
