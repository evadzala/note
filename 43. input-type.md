### 紀錄內容：

最近一直在研究原生的input
input有著許多type應付各種需求

```html
<template>
	<input @input="inputThing" />
	
	<input type="text" />
	// 可輸入文本
	
	<input type="button" />
	// 就是一個button
	
	<input type="checkbox" />
	// 複選框
	
	<input type="radio" />
	// 單選框
	
	<input type="password" />
	// 密碼，可隱藏輸入值
	
	<input type="tel" />
	// 用於輸入電話，用於手機版會出現電話鍵盤
	
	<input type="url" />
	// 用於網址，用於手機版會出現方便輸入網址的鍵盤
	
	<input type="email" />
	// 用於mail，用於手機版會出現方便輸入mail的鍵盤
	
	<input type="number" />
	// 僅能輸入數字，數值過大會顯示自然數e，可用鍵盤上下控制數值
	
</template>
```

以上只是列出常用的input type

但目前市面上有許多UI套件都相當便利
研究一下原生的方法也是很有趣
