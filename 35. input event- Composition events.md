### 紀錄內容：

因為需求須限制input只能輸入英文數字並阻擋中文

查詢後發現有compositionstart可用

研究了一下發現他有以下幾個行為:

- `compositionstart`: 輸入框內開啟輸入法，且正在拼字時觸發。
- `compositionupdate`:輸入框內開啟輸入法，且正在拼字或選字時更改了內容時觸發。
- `compositionend`: 輸入框內開啟輸入法，拼字或選字完成，正要送出至輸入框時觸發。

```html
// 用法
<template>
	<input @compositionstart="handleCompositionStart" />
</template>
<script>
	const handleCompositionStart = (e: CompositionEvent) => {
		//阻擋中文輸入法
		e.preventDefault()
	}
</script>
```
