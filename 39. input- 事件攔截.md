### 紀錄內容：

因為需求須限制input只能輸入英文數字並阻擋中文

試了幾個方案後發現都無效果

最後發現navie為了打包套件
輸入參數只能是string而不是event
所以用原生的input可避免此問題

```html
<templete>
	<input v-model="value1" @input="enOnly" />
<templete>

<script>
const value1 = ref('')

const enOnly = (event) => {
// event.target.value就是輸入所得到的值
// 判斷不是英數就用空字串取代
	event.target.value = event.target.value.replace(/[^A-Z]a-z0-9/g, '')
}
</script>
```
