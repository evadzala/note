### 紀錄內容：

在絕大多數情況下，Vue 建議使用模板語法來建立應用程式。然而在某些使用場景下，我們真的需要用到JavaScript 完全的程式設計能力。這時**渲染函數**就派上用場了。

```jsx
// 除了類型必填以外，其他的參數都是可選的
h('div')
h('div', { id: 'foo' })

// attribute 和 property 都能在 prop 中書寫
// Vue 會自動將它們指派到正確的位置
h('div', { class: 'bar', innerHTML: 'hello' })

// 像是 `.prop` 和 `.attr` 這樣的屬性修飾符
// 可以分別透過 `.` 和 `^` 前綴來加入
h('div', { '.name': 'some-name', '^width': '100' })

// 類別與樣式可以像在模板中一樣
// 用陣列或物件的形式書寫
h('div', { class: [foo, { bar }], style: { color: 'red' } })

// 事件監聽器應以 onXxx 的形式書寫
h('div', { onClick: () => {} })

// children 可以是一個字串
h('div', { id: 'foo' }, 'hello')

// 沒有 props 時可以省略不寫
h('div', 'hello')
h('div', [h('span', 'hello')])

// children 陣列可以同時包含 vnodes 與字串
h('div', ['hello', h('span', 'hello')])
```

得到的vnode 為如下形式：

```jsx
const vnode = h('div', { id: 'foo' }, [])

vnode.type // 'div'
vnode.props // { id: 'foo' }
vnode.children // []
vnode.key // null
```
