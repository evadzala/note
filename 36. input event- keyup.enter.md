### 紀錄內容：

在使用input元件時
當填寫完資訊時希望可以一鍵送出
通常使用者習慣會是按下Enter
所以在程式上可以操作其行為

例如可以用在輸入框很多的表單，節省其時間

```html
// 用法
<template>
	<input ref='input1' :value="inputA" @keyup.enter="goToInput2" />
	<input ref='input2' :value="inputB" @keyup.enter="submit" />
</template>
<script setup lang="ts">
const input1 = ref<HTMLInputElement | null>(null)
const input2 = ref<HTMLInputElement | null>(null)
const inputA = ref('')
const inputB = ref('')

const goToInput2 = () => {
// enter後會離開input1並指向input2
	input1.value?.blur()
	input2.value?.focus()
}
const submit = () => {
	// call api
}
</script>

```
