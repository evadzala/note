### 紀錄內容：

因為需求須限制input只能輸入英文數字並阻擋中文

查詢後發現CSS有ime-mode可用

```html
<input type="text" id="t2" style="ime-mode:disabled" />
```

默認值: auto

內建參數:

- auto: 不影響IME狀態
- active: 指定所有IME的輸入，即開啟本地語言輸入法，使用者仍可取消開啟IME
- inactive: 指定所有不使用IME的輸入，即開啟非本地語言，使用者仍可取消開啟IME
- disabled: 對於有焦點的元件(例如input)完全禁用IME，使用者不可開啟IME

限制: 只有[IE](http://msdn.microsoft.com/en-us/library/ms530767(VS.85).aspx?WT.mc_id=DOP-MVP-37580)與[FireFox](https://developer.mozilla.org/en/CSS/ime-mode)支援
